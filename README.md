# Postgres-Observability-Dashboard

🚀 Getting Started
1. Clone repository
git clone https://github.com/sory89/Postgres-Observability-Dashboard.git
cd Postgres-Observability-Dashboard
2. Create virtual environment
python -m venv venv
source venv/bin/activate
3. Install dependencies
pip install -r requirements.txt

docker compose up -d
