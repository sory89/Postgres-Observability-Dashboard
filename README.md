# Postgres-Observability-Dashboard

## 🚀 Getting Started

### 1. Clone repository

```bash
git clone https://github.com/sory89/Postgres-Observability-Dashboard.git
cd Postgres-Observability-Dashboard
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run with Docker

```bash
docker compose up -d
```
